# Teflon Todo App

## Features
* Skapa en todo
* Ta bort en todo
* Se antal todos kvar
* Persistance i localStorage
* Visa en meny med
  * Dölja färdiga todos
  * Radera färdiga todos
  * Tömma localStorage
